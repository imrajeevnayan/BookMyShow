# Project Roadmap

## Milestone 1: Core Experience (Now)

### Phase 1: Foundation & Discovery
*Goal: A stunning homepage that connects to backend movie data.*
- [ ] **Plan 1.1: Foundations** - Design tokens, Navbar, and Responsive Layout.
- [ ] **Plan 1.2: Discovery UI** - Hero Carousel and Movie Search/Filtering.
- [ ] **Plan 1.3: Movie Grid** - High-quality movie cards and "Now Showing" section.
- [ ] **Plan 1.4: API Bridge** - Axios setup and connection to `MovieController`.

### Phase 2: Booking Engine
*Goal: The full flow from selection to confirmation.*
- [ ] **Plan 2.1: Movie Details** - Deep-dive page into movie synopsis, cast, and trailers.
- [ ] **Plan 2.2: Seat Selection** - High-end interactive UI for selecting cinema seats.
- [ ] **Plan 2.3: Booking Integration** - Secure booking flow linking to `BookingController`.
- [ ] **Plan 2.4: Confirmation** - Booking success/failure screens and mock tickets.

### Phase 3: Identity & Administration (Future)
*Goal: Personalized profiles and admin management.*
- [ ] **Plan 3.1: Auth UI** - Login and Registration pages with validation.
- [ ] **Plan 3.2: User Profile** - My Bookings and account management.
- [ ] **Plan 3.3: Admin Portal** - Movie and Show management dashboard.
