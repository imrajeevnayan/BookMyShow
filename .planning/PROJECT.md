# BookMyShow Clone

## What This Is
A premium movie booking platform built with Spring Boot (backend) and React (frontend). It aims to provide a high-end user experience similar to major entertainment platforms.

## Core Value
Streamlining movie discovery and seat booking with a stunning, modern interface.

## Success Criteria
- [ ] Responsive, premium React frontend.
- [ ] Seamless integration with Spring Boot REST APIs.
- [ ] Functional seat selection and booking flow.
- [ ] Real-time-ish movie search and filtering.

## Requirements

### Active
- [ ] Initialize premium React/TS frontend with Vite.
- [ ] Design and implement Home Page (Now Showing, Featured Carousel).
- [ ] Design and implement Movie Details Page (Synopsis, Cast, Trailer).
- [ ] Seat Selection Interface (Interactive theater layout).
- [ ] Booking Integration (Connecting frontend to Backend `BookingController`).
- [ ] User Authentication (Connecting to Backend `UserController`).

### Out of Scope
- [ ] Real Payment Gateway integration (using mocks for now).
- [ ] Social Login (Google/Facebook auth).
