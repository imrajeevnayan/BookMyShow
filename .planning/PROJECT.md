# BookMyShow Clone

## What This Is
A premium movie booking platform built with Spring Boot (backend) and React (frontend). It aims to provide a high-end user experience similar to major entertainment platforms.

## Core Value
Streamlining movie discovery and seat booking with a stunning, modern interface.

## Success Criteria
- [x] Responsive, premium React frontend.
- [x] Seamless integration with Spring Boot REST APIs.
- [x] Functional seat selection and booking flow.
- [x] Real-time-ish movie search and filtering.

## Requirements

### Active
- [x] Initialize premium React/TS frontend with Vite.
- [x] Design and implement Home Page (Now Showing, Value Props, Footer).
- [x] Design and implement Movie Details Page (Synopsis, Cast, Interactive Booking Card).
- [x] Seat Selection Interface (Interactive theater layout, dynamic summary bar).
- [ ] Finalize Backend/Frontend integration for all edge cases.
- [ ] User Authentication (Connecting to Backend `UserController`).

### Out of Scope
- [ ] Real Payment Gateway integration (using mocks).
- [ ] Social Login (Google/Facebook auth).
